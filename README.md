# dating_API
Created Back-end API'S For Dating App Project Given By Bheema Info-tech 


## Setup
1. Create MySQL database using `dating_app_database.sql`
2. Update DB credentials in settings.py
3. Install dependencies:
   pip install -r requirements.txt
4. Run migrations:
   python manage.py migrate
5. Start server:
   python manage.py runserver

## API Testing
Import Postman collection file:
Dating_App_Backend.postman_collection.json
